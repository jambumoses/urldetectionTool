FAQ

Q: How does urlscan.io work?

A We use the Google Chrome browser in Headless Mode to browse to the URLs submitted by users. We record the interaction of the page with the Internet and after the page has finished loading, we annotate the results with additional data sources

Q: Does urlscan.io show whether a website contains malware or phishing attempts?

A Yes, we have some basic mechanisms for determining whether a website contains malicious content. Our proprietary phishing detection mechanism tracks 500 popular brands and can identify phishing or impersonation attempts of these brands. We do record file downloads, but we do not detect whether a downloaded file is malicious, e.g. a malicious executable.
